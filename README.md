# Lifters Club excersise REST api

Built with Nest JS this is intended to use as a sort of exercise roladex

## Project setup

```bash
$ npm install
```

## Compile and run the project

After filling out the `.env` and the `docker-compose.yml` files

```bash
$ docker-compose up
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
